AI 日报（2026-03-11）

今日重点：
- OpenAI 提升前沿 LLM 的指令层级能力：OpenAI 发布 IH-Challenge，核心是让模型更清楚地区分高可信与低可信指令，从而增强安全可控性，并提升对 prompt injection 的抵抗力。
- ChatGPT 上线更适合学习数理的交互式讲解：ChatGPT 开始加入更强的可视化与互动式讲解能力，数学和科学学习场景明显更像“交互教具”而不只是问答框。
- NVIDIA 解释如何构建面向 AI 的开放数据：文章聚焦 NVIDIA 如何建设开放数据体系，重点价值在于训练数据工程、质量治理和生态协作。
- Gemini 在 Google Sheets 中拿到更强的数据分析能力：Google 正在把 Gemini 从“表格助手”往“自然语言驱动的数据分析接口”推进，办公 AI agent 味道越来越浓。
- GitHub：AI 不再只是文本接口，执行才是新界面：GitHub 的判断很明确：下一阶段 AI 产品的核心不是聊天，而是进入可执行工作流，把 agent 能力嵌进应用本身。
- 16 个开源 RL 库的经验总结：训练系统开始卷工程效率：这篇总结的重点不在算法花样，而在异步训练、吞吐与工程权衡，说明训练基础设施仍然是高价值方向。
- Hugging Face Hub 推出 Storage Buckets：Hugging Face 正在继续下探到更底层的基础设施层，说明它的目标不只是模型托管，而是更完整的 AI 开发平台。
- Simon Willison：AI 应该帮助我们写出更好的代码：如果引入 coding agent 后代码质量下降，问题不该怪在“用了 AI”，而该回到流程、评审和反馈机制本身。
